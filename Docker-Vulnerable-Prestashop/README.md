## Lancement

```bash
$ bin/start
```

## Accès aux containers

MySQL:

> Depuis l'hôte sur le port 3307
> > `$ mysql -uroot -padmin -h localhost --port 3307`

> Depuis un container dans le réseau avec l'url `docker-prestashop-mysql`
